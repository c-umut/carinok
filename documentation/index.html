Installation

We need a couple of applications to run the Quiety to our local:

Node.js:

It is used to run the build processes. Download from the NodeJs from the official site and install it on your device according to your Operating System (nodejs.org)

Check NodeJs Version:

  • Open the Terminal
  • Run: node -v or node --version
  • If you get the version then the installation of NodeJs is done
$ node --version
$ v14.18.1

React Packages Install:

To install the theme, you have to install React then go to the theme root directory where package.json located. Now install packages use this commend. Before running the command, make sure that you have nodejs installed to your machine.


$ npm install

it will install the packages. Make sure that there is no error.

Configuration

package.json: Make sure that you have package.json file and check that you have the below lines under the scripts.

... "scripts": {
"start": "react-scripts start",
"build": "react-scripts build",
"test": "react-scripts test",
"eject": "react-scripts eject"
},
...

How to Run?

After install process now you can run local server- local server port is 'http://localhost:3000' For development start use this commend npm start


$ npm start

How to Build?

Only build the project:


$ npm build

Run static server (optional)

If you want to run the build project then we need to follow the following steps:

First install serve:


$ npm install -g serve

Then build the project:


$ npm run build

Finally, run the project:


$ serve -s build

If you want to know more about static deployment then please visit Create react app deployment
